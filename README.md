# UPoc2026

このリポジトリは、2026年度 UPoc の企画・開発用です。  
開発ブランチは `UPoc2026` 想定で進行します（現在の作業ブランチで反映）。

## 企画アイデア
### ペットボトルキャップを投入すると絵が完成するリサイクル参加型アート装置

- **対応テーマ**: 誰でも楽しめるイベント体験の創出 / 調布駅前広場でのイベント企画支援 / ゼロカーボンシティの実現に向けた市民等の行動促進
- **想定導入先**: 調布駅前広場・調布市内商業施設・大学構内

## 概要
街角に設置したボックスにペットボトルキャップを投入すると、カメラで色を検出して1ピクセルとして記録し、1,000本分で大型ドット絵を完成させます。  
進捗は Web / 現地ディスプレイにリアルタイム表示され、「自分の1ピクセル」が見える体験でリサイクル行動を促進します。  
色分布データを AI が分析して次回アート配色を提案し、個人貢献証明書（投入位置可視化シェア画像）で継続参加を促します。

## ターゲットユーザ
- 子供（5〜12歳）とその親・家族、幅広い市民参加者（学生・社会人・高齢者を含む）
- リサイクル率向上を目指す調布市・自治体
- 集客・PR を強化したい商業施設・大学

## 課題と解決
- 現状: 「リサイクルしなさい」という義務訴求が中心で、子供の自発行動が起きにくい
- 解決: 「みんなで絵を完成させる」達成感・参加感を起点に行動変容を設計

## 調布市との連携ポイント
- 調布駅前広場での設置により通勤・通学動線での習慣化
- テーマを「調布の風景」「鬼太郎」等の地域性と接続
- 完成時の除幕イベントを市主催企画化し、イベント創出とゼロカーボン施策を同時推進

## 独自性
1. 義務感ではなく達成感でリサイクルを促進
2. 投入色分布を使った AI の次テーマ配色提案
3. 「あなたのキャップはここ」の個人貢献証明書生成
4. タブレットデモ → Raspberry Pi 常設版への段階的実装

## 機能
### ハードウェア
- キャップ投入口付きボックス
- Raspberry Pi + PiCamera2 + OpenCV による色検出
- 投入直後に色・ピクセル番号を表示
- QR でリアルタイム進捗ページへ誘導

### Webアプリ
- HSV 変換で色をピクセルデータ化
- ドット絵をリアルタイム更新
- 「あと○本で完成」カウントダウン
- 投入番号によるピクセル位置トラッキング
- 完成後に次テーマを地域投票

### AIコンポーネント
- 色分布ヒストグラム分析による次回配色提案
- 個人貢献証明書（シェア画像）自動生成

### コミュニティ施策
- 匿名/ニックネームランキング
- 達成バッジ（例: 10本投入）
- SNS シェア動線

## 技術構成
| 技術 | 役割 | 選定理由 |
|---|---|---|
| Raspberry Pi 4 + PiCamera2 | ハードウェア制御・色検出 | 小型・低コストで常設しやすい |
| OpenCV + HSV | キャップ色の自動分類 | 照明変動に比較的強く軽量 |
| FastAPI | イベント記録・データ管理 | Python で Pi 連携と API を統一 |
| React + WebSocket | リアルタイム表示 | 投入ごとの即時更新に適合 |
| PostgreSQL | 投入履歴・ピクセル管理 | 投入番号×座標×色の管理に適合 |
| LLM（画像生成AI） | 次テーマ提案 | 色分布から配色案を生成可能 |
| Canvas API | 証明書画像生成 | ブラウザで可視化画像を生成可能 |

## デモシナリオ（15〜20分）
1. タブレット + 投入台を設置
2. キャップをかざして色検出
3. 色とピクセル番号を即時表示、ドット絵更新を確認
4. QR から進捗・残数をスマホで確認
5. 個人貢献証明書を生成して提示
6. AI 配色提案と地域投票画面を紹介

## MVP デモ実装（FastAPI + Canvas）

このリポジトリには、**キャップ投入 → ドット絵更新 → 個人証明書生成**までを体験できる
簡易デモを実装しています。実機のカメラ検知は色入力に置き換えています。

### できること
- キャップ色を投入してドット絵をリアルタイム更新
- 「あと◯本で完成」のカウントダウン表示
- 投入番号から個人貢献証明書画像を生成
- 投入色分布から AI 提案（簡易ロジック）を表示
- 地域投票（テーマ投票）とランキング表示

### 起動方法
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

ブラウザで `http://localhost:8000` を開くとデモ画面が表示されます。

### ディレクトリ構成
```
app/main.py        # FastAPI バックエンド（WebSocket + API）
static/index.html  # デモ UI
static/app.js      # Canvas 描画・リアルタイム更新
static/styles.css  # UI スタイル
```
